Задания к работе №1 по фундаментальным алгоритмам.



1.	Через командную строку задается число. Вывести на экран числа в пределах 100, кратные указанному. 
Если таковых нету – вывести соответствующее сообщение.

2.	Пользователь вводит число. Программа должна определить является ли введенное число простым или составным. 
Программа завершает свою работу только, если было введено число 0. 

3.	Текстовый файл содержит некоторую информацию. Необходимо перезаписать файл, исключая цифры. 

4.	На вход программе подаются коэффициенты квадратного уравнения. Необходимо вывести решения этого уравнения при всех 
возможных значениях параметров.

5.	Напишите программу, которая считывает два ненулевых целых числа, после чего определяет, кратно ли первое второму, 
и выводит информацию на экран.

6.	Напишите программу, которая считывает 7-значное число, разделяет его на отдельные цифры и выводит 
отдельно каждую цифру, разделяя их пробелом.

7.	Написать программу, которая считывает ненулевое число (для простоты не больше 10) и выводит числа от 1 до 10,
возводя во все степени до введенного числа.

8.	Написать программу, которая считывает 3 ненулевых значения и проверяет могут ли они быть сторонами 
прямоугольного треугольника.

9.	Напишите программу, которая на вход принимает ненулевое число и суммирует все числа от 
1 до указанного числа включительно.

10.	Написать программу для вычисления факториала числа.

11.	На вход программе, через аргументы командной строки, подается файл. Необходимо прочитать файл и 
в файл трассировки (выходной файл) написать сколько раз в каждой строче встречаются буквы.

12.	На вход программе, через аргументы командной строки, подается файл. Необходимо прочитать файл и в файл трассировки 
(выходной файл) написать сколько раз в каждой строчке встречаются символы, отличные от цифр, букв и символа пробела.

13.	Дан файл, содержащий некоторую информацию, которая представлена в трех столбцах. Необходимо перезаписать этот файл 
таким образом, чтобы первый столбец стоял на месте второго, второй – на месте третьего, а третий – на месте первого.

14.	На вход программе, через аргументы командной строки, подается файл. Необходимо перезаписать файл, заменяя символы,
отличные от цифр, их ASC-II кодом.

15.	На вход программе подается два файла. Написать третий файл, где на нечетных позициях будут стоять лексемы из 
первого файла, а на четных – из второго.

16.	На вход программе подается некоторый файл. Перезаписать его в новый файл таким образом, чтобы в каждой 5-ой 
лексеме все символы были заменены на эквивалентные им ASC-II коды, а в каждой второй лексеме все буквы были 
переписаны в строчные.

17.	В текстовом файле находятся числа записанные в разных системах счисления, при этом информация о конкретной 
системе счисления для каждого числа утеряна. В файле числа разделены произвольным количеством разделителей. 
Напишете программу, которая для каждого числа определяет минимальную систему счисления, в которой представление 
этого числа корректно и в выходной файл выводит число, определенную для него систему счисления и представление 
этого числа в 10-ой системе счисления.

18.	Заполнить массив фиксированного размера случайными числами. Реализовать поиск максимального 
и минимального элементов, поменять местами максимальный и минимальный элементы массива.

19.	Даны два одномерных массива случайных чисел А и В. Сформировать из них массив С, где к каждому элементу 
из А добавить ближайший к нему элемент из массива В.

20.	Пользователь вводит систему счисления и затем числа в этой системе счисления. Окончанием ввода 
является ввод слова Stop. Найдите среди введенных чисел максимальное. Переведите найденное максимальное 
число в 9, 18, 27 и 36-ые системы счисления.
